Settings:
Controls how fast new emails/reminders are detected and notifications will appear. Lower value means faster reaction. Minimum allowed value is 1.
When reminders are not dismissed/snoozed a reminder notification will keep showing up every 'Reminder snooze time' seconds. Set this to zero to disable it.
Toggle for updating the favicon.
Toggle for updating document title.
In case number of unread emails is not showing at all or you only want certain folders to be included you can use this to customize what is counted as unread email. You need to have html/css knowledge in order to use it. If any value is entered here it will be used as an override to the existing code/behaviour. As an example for OWA 2010 this css selector "[fldrnm='Inbox'] + #spnUC #spnCV" will only count unread emails from Inbox folder.
In case reminders notifications are not working you can use this to customize the css selector used to detect reminders. You need to have html/css knowledge in order to use it. If any value is entered here it will be used as an override to the existing code/behaviour. As an example for OWA 2010 this css selector "#lnkRm > #spnRmT.alertBtnTxt" is used to detect reminders.
