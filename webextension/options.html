Settings:
Controls how fast new emails/reminders are detected and notifications will appear. Lower value means faster reaction. Minimum allowed value is 1.
When reminders are not dismissed/snoozed a reminder notification will keep showing up every 'Reminder snooze time' seconds. Set this to zero to disable it.
Toggle for disabling notifications. When checked, new emails, reminders or chat events won't trigger the display of notifications anymore, but the update of favicon and document title will continue (if those options are active)
Toggle for updating the favicon. When checked, the total count of all emails, reminders and chat events will be displayed as a number on the webpage icon.
Customize favicon background color.
Toggle for updating document title. When checked, the webpage document title will be updated with the counters of emails, reminders and chat events
If the number of unread emails is not showing at all or you only want certain folders to be included you can use this to customize what is counted as unread email. You can try a predefined option or define one yourself selecting the "Custom" option and entering in the edit box the css you want (some html/css knowledge might be needed).
If reminders notifications are not working properly, you can use this to customize the css selector used to detect reminders. You can try a predefined option or define one yourself selecting the "Custom" option and entering in the edit box the css you want (some html/css knowledge might be needed).
If chat notifications are not working properly, you can use this to customize the css selector used to detect chat messages. You can try a predefined option or define one yourself selecting the "Custom" option and entering in the edit box the css you want (some html/css knowledge might be needed).
